## 切片说明（以PrusaSlicer为例）
导风口需按以下条件进行特殊设置：
1. 0.4打印喷嘴，需要将“最小轮廓数”改为2，以便搭桥起点下方不悬空
2. （填充图案为星型时）“填充率”至少设为40%
3. 取消“自动生成支撑”，导风口可无支撑打印

贴墙端需按以下条件进行设置：
1. （填充图案为星型时）“填充率”至少设为40%

切片摆放方向：螺丝孔洞与Z轴平行。

## 改进空间
1. 取消3010风扇安装座，直接在开口平面上安装3015风扇，理论上与散热器之间有0.4mm距离
2. 贴墙端固定件可使用热熔滚花螺母
